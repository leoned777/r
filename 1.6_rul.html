Команды

5.1 Команда → "Топ" показывает активных участников дня.

5.2 Команда → "Топ" показывает активных участников дня.

5.3 Команда → "Заметки" показывает все существующие заметки.

5.4 Команда →"Кто я" показывает информацию о вас.

5.5 Команда → "клан создать (название) " создание своего клана. "+клан" вступить в клан.

5.6 Команда → "Кто админ" чтобы узнать список администраторов чата.

5.7 Команда → "фарма" дает койны.

5.8 Команда → "+девиз (название)" добавить свой дивиз.

5.9 Команда → "кружок создать (название)" добавить свой кружок.

5.10 Команда →"+ник (название)" изменить/добавить свой ник.
